What is Data Science?

In the ML section we discussed how computer are feed with data to learn the patterns and when similar kind of data comes along the computer automatically performs the task. Now organising data, finding patterns from the data, making it structured, trying to learn from it is an essential task. All this tasks are performed by Data Science.

Statisticians before used to work with data and analysis. But all this required structured data. Recent era see data which are all not so well strucutred (semi-structured) and unstructured. Also storing data was also a big challenge back in 2010. Now new frameworks like Hadoop, takes care of the storage part, comes the challenge of analysis.

Mathematical Techniques, ML and Advanced Algorithms and Programming Techniques helps in this process. This is what constitutes Data Science.
